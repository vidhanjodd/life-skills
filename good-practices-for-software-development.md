# Good Practices for Software Development

## Which point(s) were new to you?

- The concepts of over-communicating with the team.
- Asking clear and detailed questions when stuck.
- Using tools like screenshots or code snippets to explain issues.

## Which area do you think you need to improve on? What are your ideas to make progress in that area?

I believe that my communication when I face issues needs to be improved. In order to improve this, I will make an effort to clearly describe my issue, mention what I have already tried, and share appropriate details or screenshots so that others can assist me more quickly. Additionally, I'll make an effort to stay more focused while working by reducing distractions.
